AWS

AWS (Amazon Web Services) est une plateforme de cloud computing proposée par Amazon. Elle offre une grande variété de services pour stocker et traiter des données, exécuter des applications et des services, et déployer des infrastructures à l'échelle mondiale.

AZURE

AZURE est la plateforme de cloud computing de Microsoft. Elle permet de déployer, de gérer et de mettre à l'échelle des applications et des services, ainsi que de stocker et d'analyser des données sur des serveurs Microsoft à travers le monde.

VMware

VMware est une plateforme de virtualisation qui permet de créer des machines virtuelles pour exécuter des systèmes d'exploitation et des applications sur des ordinateurs physiques. Elle offre également des solutions de cloud computing pour les entreprises.

Linux

Linux est un système d'exploitation open source qui peut être utilisé pour le cloud computing. Il offre des solutions de virtualisation, de stockage et de réseau pour les entreprises qui souhaitent créer et gérer leur propre infrastructure de cloud computing.
